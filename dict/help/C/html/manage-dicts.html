Manage dictionaries

  • StarDict Manual V3.0.6

This section of the manual describes the Manage dictionaries dialog that allows you to view and organize all dictionaries available in StarDict. You may work with the following types of dictionaries: normal (index-based) dictionaries, virtual plugins, network plugins, network dictionaries, tree dictionaries. The dialog consists of 4 tabs: Section 5.1 ― Dict list tab, Section 5.2 ― Manage dict tab, Section 5.2 ― Manage dict tab, Section 5.3 ― Tree dictionaries tab. Each tab is described in details below.

  • 5.1. Dict list tab
  • 5.2. Manage dict tab
  • 5.3. Tree dictionaries tab
  • 5.4. Network dictionaries tab

5.1. Dict list tab

Dict list tab allows you to view and reorder a list of all dictionaries of the following types: normal (index-based) dictionaries, virtual plugins, network plugins. Dictionaries of these types may be organized into groups. To work with dictionaries groups, switch to Section 5.2 ― Manage dict tab. This tab shows all dictionaries that are known to StarDict, irrespective of whether they are used in some dictionary group or not.

Figure 6 Dict list tab

Columns of the dictionary list as self-explaining. Dictionary name ‒ name of the dictionary for a normal dictionary and the name of plugin for plugins. Word count ‒ number of word in a normal dictionary. For plugins this column always contains 0. Type ‒ type of the dictionary Local for normal dictionaries, Virtual for Virtual plugins, Network for Network plugins.

You may reorder dictionaries in the list with buttons on the right or using drag and drop technology. Note that you can select only one dictionary at a time. This list specifies the default order of dictionaries for Add dictionary dialog. That dialog is used to add new dictionaries to groups, see Section 5.2 ― Manage dict tab. The order of dictionaries is saved between application restarts. Note, however, that virtual and network plugins are always appear at the end of the list. Their order is not saved, StarDict saves only the order of normal dictionaries.

To get detailed information about a dictionary, double-click on a list item. Dictionary information dialog appears. Here you may see dictionary name, word count, author, email and other fields.

5.2. Manage dict tab

This tab allows you organize all dictionaries listed on Section 5.1 ― Dict list tab in groups. By default StarDict has one group including all dictionaries. You may create new groups to organize your dictionaries better. Switching dictionary group allows you to quickly change a set of dictionaries where StarDict searches words. See Section 2.2 ― StarDict Main window, Left bar for information about how to switch dictionary group. For example, you may organize your dictionaries by a language pairs: English to Chinese, Chinese to German, German to Russian, etc.

Figure 7 Manage dict tab

Each group consists of two subgroups: Query Dict and Scan Dict. The first subgroups is used for queries in the Main window, while the second is used in Scan mode. Subgroups contain lists of dictionaries. The order of dictionaries this dialog correspond to the order of results in Main window or Floating window, depending on subgroup. Note, however, that results from Network plugins are always added at the bottom. That is because response from a Network plugin is normally asynchronous and comes with unpredictable delay.

Dictionary list is organized in a tree, you may collapse groups and subgroups to save space. You may reorder dictionary groups and dictionaries inside a subgroup using buttons at the right or using drag and drop technology. Note, that can select only one item at a time. You cannot move dictionaries between groups and subgroups.

To get detailed information about a dictionary, double-click on a row, or select a row and click the button with a bulb, or click Show information in context menu.

To remove a dictionary from a subgroup, or to remove a group, select that list item, then click on the button with a trash can. Note that removing a dictionary does not affect the corresponding file on the hard disk. Clicking the remove button on a subgroup, removes all dictionaries from that subgroup.

Each dictionary in the list can be independently enabled/disabled. Disabling a dictionary has the same effect as removing it from the list, it will not be used to search words when the corresponding dictionary group is selected. Disabled dictionary retains its position in the list, you can easily enable it later. In the context menu you may find Select all, Unselect all items. They select (unselect) all dictionaries in a subgroup.

To add new dictionaries into subgroup, you need to select a subgroup or a dictionary inside it, and click the button with a plus sign. Add dictionary dialog appears. Here you see a list of all dictionaries that are not yet added to the selected subgroup. The order of dictionaries is the same as on Section 5.1 ― Dict list tab. You may select any number of items in this list. Some columns are clickable. Clicking on a column title sort the list by that column. Sorting by Path column is especially useful if your dictionaries are organized in directories corresponding to dictionary groups. After sorting all dictionaries from the same directory will be located in adjacent rows.

Figure 8 Add dictionary dialog

To rename a group, select it, then click on the selected row again. Now you may edit the group name in place.

5.3. Tree dictionaries tab

On this tab you may see a list of all tree dictionaries. Tree dictionaries are not used when you type a word into the Main window or use the Scan feature. Tree dictionaries are shown on a tab of its own in the Main window, see Section 2.2 ― StarDict Main window, Tree button on the Left bar.

Figure 9 Tree dictionary tab

You may reorder dictionaries with the buttons on the right or using drag and drop technology.

To get detailed information about a dictionaries, double-click on it or select Show information from the context menu.

You may disable any dictionaries using a checkbox in the first column. In the context menu you may find Select all, Unselect all items that selects/unselect all dictionaries in the list.

5.4. Network dictionaries tab

Network dictionaries tab contains a list of network dictionaries. As the name implies, network dictionaries are not installed on your machine, they reside on a server in Internet. StarDict uses a custom protocol to query remote dictionaries and get results. Before you can add dictionaries into this list, you need to register on the server, see Section 4.2.1 ― Net Dict.

Figure 10 Network dictionaries tab

You may reorder the dictionaries using the buttons on the right of the dialog or using drag and drop technology.

To get detailed information about a dictionary, double-click on it.

It's perfectly normal that StarDict does not react immediately to your activity. Many action in this dialog require access to the remote server. The response comes with unpredictable delay.

To add new dictionaries, click on the Add button. Browse dictionaries dialog appears.

In Browse dictionaries dialog, expand dictionary groups to see dictionaries. To get information about a dictionary, select a dictionary, then click the Information button. To add a dictionary, select a dictionary on the list, then click the Add button.

Figure 11 Browse dictionaries dialog

To remove dictionaries from your list, click on the Remove button.
